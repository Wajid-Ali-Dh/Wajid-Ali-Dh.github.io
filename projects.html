Projects

Assistive Feeding System

Designed and developed a 6-DOF robotic manipulator to feed patients with limited mobility, integrating face detection and 3D facial landmark localization using MediaPipe and OpenCV. The system features a vision module for mouth tracking, a scooping and delivery mechanism, and is fully embedded on a Raspberry Pi with wireless control. Offers both passive and active feeding modes for patient assistance.

Key Technologies Used: Python, Raspberry Pi, MediaPipe, OpenCV, Monocular Depth Estimation, 3D Printing (PLA+), FDM, Motor Control

AI-Based Defect Detection and Adaptive Manufacturing Correction System

Built a real-time YOLOv8-based computer vision system to detect manufacturing defects such as scratches, tool wear, and misalignment from images, videos, and live camera feeds. Integrated an adaptive decision module with simulated IMU/encoder data for process correction or pausing. Developed an interactive Streamlit dashboard for live monitoring and smart manufacturing simulations.

Key Technologies Used: Python, YOLOv8, OpenCV, Streamlit, Sensor Fusion, IMU Simulation, Encoder Data Processing

Drone Path Planning and Dynamic Obstacle Avoidance using RRT*

Simulated autonomous UAV navigation in a 2D environment using the RRT* algorithm for optimal path planning in dynamic, cluttered spaces. Features challenging maze-like obstacles, continuous path refinement, and full visualizations of the planning process from tree growth to goal execution. Implemented in Python with animated simulations for clear demonstration.

Key Technologies Used: Python, RRT*, Path Planning Algorithms, NumPy, Matplotlib, ImageIO

Mobile Human Intrusion Detection System

Developed an NVIDIA Jetson-powered vision system for tunnel inspection, capable of detecting unauthorized humans, animals, vehicles without number plates, and unsafe maintenance practices (missing helmets or vests). Validated the custom object detection models on 60,000 images and 600 videos with dynamic lighting conditions, enabling real-time alerts and automated evidence capture with 30 seconds pre- and post-event recording. Designed for deployment across multiple tunnel entry and exit points as part of a government infrastructure security project.

Key Technologies Used: NVIDIA Jetson, Python, YOLO-based Object Detection, OpenCV, Real-time Video Processing, Edge AI, Battery-powered Systems
